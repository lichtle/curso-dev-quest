CSS Externo

Subtítulo 1

O CSS externo é o mais recomendado pois todo o CSS fica contido em um só arquivo (geralmente nomeado style.css), evitando repetição de código, proporcionando organização e eficiência caso seja necessário fazer qualquer alteração em uma ou mais páginas de um site.

Subtítulo 2

Ao inspecionarmos o código fonte dessa página é possível ver a tag <link> dentro do head, a qual possui dois atributos: rel e href, onde rel indica o tipo de arquivo (Style Sheet/CSS) e o href indica o caminho em que o arquivo se encontra. O arquivo CSS linkado segue a mesma sintaxe da tag <style>.
