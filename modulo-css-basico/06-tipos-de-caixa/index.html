Box Model (modelo de caixas)

Os elementos no CSS se comportam como caixas.

Dito isso, temos dois tipos de caixa: block level e inline level. A maioria dos elementos tem comportamento display: block;. Por exemplo, o heading acima, o atual parágrafo e o parágrafo abaixo têm esse comportamento e estão demarcados com uma borda verde.
