CSS Interno

Subtítulo 1

O CSS interno é mais recomendado que o inline pois ele reduz códigos repetitivos, porém uma forma ainda melhor é criar um arquivo próprio para o CSS. Dessa forma podemos alterar o estilo de mais de uma página do site sem ter que copiar o mesmo código e colar em todas as webpages. Basta alterar no arquivo CSS.

Subtítulo 2

Ao inspecionarmos o código fonte dessa página é possível ver a tag <style> dentro do head, permitindo aplicar a mesma estilização em mais de um elemento sem que a gente precise repetir o mesmo código em cada elemento.
