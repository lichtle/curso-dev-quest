:nth-last-child

Diferente da :nth-child, esta pseudo-classe começa sua contagem do último pro primeiro filho.

:only-child

Estiliza determinado elemento filho somente quando ele não tem elementos irmãos, como esse parágrafo solitário dentro de uma div :( neste caso somente ele, dentre todos os parágrafos da página, está sendo estilizado com a cor rosa.
