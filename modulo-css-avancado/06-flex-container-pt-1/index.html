Flex Container: direction, wrap e flow

No display do tipo flex trabalhamos o comportamento dos containers e dos elementos que estão dentro deles.

flex-direction

No exemplo abaixo o valor da propriedade flex-direction é row-reverse:

1
2
3

flex-wrap

No exemplo abaixo o valor da propriedade flex-wrap é wrap. Diminua o tamanho da janela do navegador para visualizar a mudança:

3
4
5
6

flex-flow

É a junção das duas propriedades acima, como uma shorthand. No caso abaixo o valor do flex-flow é row-reverse wrap:
